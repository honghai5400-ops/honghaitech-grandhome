# Project: HongHaiTech GrandHome - Property Website

## Overview
A premium Real Estate / Property website built with React and Vite, based on a professional Figma design. The site aims to provide a high-end experience for users searching for their dream homes.

## Tech Stack
-   **Core**: React (Vite)
-   **Logic**: JavaScript
-   **Styling**: TailwindCSS (v4)
-   **Typography**: Inter (Google Fonts)

## Design System

### Colors
-   **Primary**: `#1E4AE9` (Vibrant Blue)
-   **Surface/Background**: `#FFFFFF` (White)
-   **Section Accents**: `#F8F9FA` (Off-white / Light Gray)
-   **Text Primary**: `#1A1A1A` (Dark Gray/Black)
-   **Text Secondary**: `#666666` (Medium Gray)
-   **Rating/Accents**: `#FFD700` (Yellow)

### Components
-   **Header**: Sticky navigation with glassmorphism or clean white background.
-   **Hero**: Dynamic property search interface.
-   **PropertyCard**: Stats for beds, size, rooms, and location.
-   **ValueCard**: Icons and text for "Why Choose Us."
-   **TestimonialCard**: User ratings and feedback.

## File Structure (Planned)
- `/src/components`: UI components (Header, Footer, PropertyCard, etc.)
- `/src/sections`: Page sections (Hero, Recommendation, Testimonials, etc.)
- `/src/styles`: CSS modules or global `index.css`
- `/src/assets`: Images and icons

## Key Features
- Responsive property listings grid.
- Integrated search and filtering.
- Client testimonial sliders.
- Blog/News section for real estate updates.

## Development Rules
- Use HSL for dynamic color variations.
- Ensure all components are responsive.
- Maintain a premium feel with subtle animations and generous whitespace.
